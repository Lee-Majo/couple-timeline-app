# 💕 Couple Timeline App

A simple web app that allows users to view a timeline of special memories with images and descriptions.

## 🚀 Features
- View memories in a vertical timeline
- Images with each memory
- Clean and simple UI
- View-only mode for public demo

## 🛠 Tech Stack
- Node.js
- Express
- SQLite
- HTML, CSS, JavaScript
